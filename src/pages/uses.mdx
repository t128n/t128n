## Dev Env

- [mise](https://mise.jdx.dev/) - dev-env management
- [ghostty](https://ghostty.org/) - terminal emulator
- [zed](https://zed.dev/) - code editor

### AI/LLM

- [opencode](https://opencode.ai/) - AI agent harness
- [π (pi)](https://pi.dev/) - minimal AI agent harness (& sdk)
- [baml](https://boundaryml.com/) - dsl for LLM inference pipelines

### Python

- [uv](https://docs.astral.sh/uv/) - Python environment and package manager
- [ruff](https://docs.astral.sh/ruff/) - python linter & formatter

### JS/TS

- [bun](https://bun.sh/) - fast all-in-one JavaScript toolkit
- [pnpm](https://pnpm.io/) - fast JavaScript package manager
- [vite](https://vite.dev/) - modern frontend tooling
- [oxfmt](https://oxc.rs/docs/guide/usage/formatter.html) - fast modern formatter
- [oxlint](https://oxc.rs/docs/guide/usage/linter) - fast modern linter
- [vitest](https://vitest.dev/) - testing framework
- [astro](https://astro.build) - modern static site generator
