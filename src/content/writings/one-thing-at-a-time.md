[Please stop multitasking. I’m begging you. Please.](https://jason.energy/context-switching/)

I catch myself multitasking way too often. Dozen different windows open, responding to messages and switching
tasks like I'm some CPU handing out time slices. Grinding through the day, but still only making marginal progress on
everything touched.

## Actionable advice

### PRUNE

1. **P**ut all tasks on a list
2. **R**ank by priority
3. **U**npack big tasks into smaller estimates
4. **N**ote a clear goal for each task
5. **E**liminate anything unnecessary

### FLOW

1. **F**ix on one single task
2. **L**ock a 90-minute timer[^1]
3. **O**ff. close all distractions
4. **W**ork until the timer ends

[^1]: 90-minutes are a good default. Long enough to get into a flow state; not so long that people start filing a missing person report.
