[make-interfaces-feel-better](https://skills.sh/jakubkrehel/make-interfaces-feel-better/make-interfaces-feel-better) by [Jakub Krehel](https://jakub.kr/).

I stumbled upon this skill set by accident some time ago. It should be a must-have for any frontend [agentic engineering](https://simonwillison.net/guides/agentic-engineering-patterns/what-is-agentic-engineering/) workflows.

It can be easily added to any project using [skills.sh](https://skills.sh/).

```sh frame="terminal" title="pnpm"
pnpm dlx skills add jakubkrehel/make-interfaces-feel-better
```

<br/>

```sh frame="terminal" title="bun"
bunx skills add jakubkrehel/make-interfaces-feel-better
```

The supporting [blog article](https://jakub.kr/writing/details-that-make-interfaces-feel-better) even has some interactive demos showcasing best practices.
